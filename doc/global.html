cbuf_clone(cbuf) → {CircularBuffer}

Clone a circular buffer. Note that the result is not a 'deep' copy, the DATA array is cloned using slice.
Note that cloning a 'subclass', i.e. a circular buffer with extra data properties, will usually work using CBUF.clone but this should be verified. E.g. an extra array data property will need extra code to clone its contents.

cbuf_create(capacity) → {CircularBuffer}

Create a new circular buffer (FIFO queue) with a fixed capacity. At no point can the circular buffer contain more than capacity elements.
Although not available as a direct property of the buffer -- the only such properties are DATA, POP and BEGIN -- it can easily be accessed using the expression c[CBUF.DATA].length where c is the circular buffer.
The function also seals the DATA array, so it cannot be extended.
Parameters:
Name Type Description
capacity integer the size of the DATA array, i.e. the maximal capacity. Must be strictly positive, obviously (but this is not verified by the implementation).
Source:
  • cbuf.es7, line 65
Returns:
A newly created empty circular buffer with a capacity equal to the parameter
Type
CircularBuffer
Example
import CBUF from 'cbuf';
 let c = cbuf_create(10);
 // postcondition: 
 assert( c[CBUF.POP] === 0 && c[CBUF.BEGIN] === 0 && 
   c[CBUF.DATA].length === capacity &&
   foreach 0 <= i < c[CBUF.DATA].length : c[CBUF.DATA][i] === undefined
   );

Example
import CBUF from 'cbuf';
 let c = cbuf_create(3);
 let x = { 'a', 'b', 'c', 'd' };
 assert(CBUF.push(cbuf,x[0])); // 'a'
 assert(CBUF.push(cbuf,x[1])); // 'b'
 assert(CBUF.push(cbuf,x[2])); // 'c'
 const u = CBUF.shift(cbuf); // => 'a'
 assert(u === x[0]);
 assert(CBUF.push(cbuf,x[3])); // 'd'
 assert(cbuf[CBUF.BEGIN] === 1);
 assert(cbuf[CBUF.POP] === 3);
 assert(CBUF.full(cbuf));

 let q = [];
 for (const z of CBUF.iterable(cbuf)) 
   q.push(z);
 // q === [ 'b', 'c', 'd' ];

 // alternative
 q = [];
 let p = 0;
   while (p < cbuf[CBUF.POP])
     q.push(cbuf[CBUF.DATA][(CBUF.inc(cbuf, cbuf[CBUF.BEGIN] + p++)))
 // q === [ 'b', 'c', 'd' ];

cbuf_last(cbuf) → {Object}

Retrieve the last element (most recently added in FIFO terms) from the circular buffer. If the buffer is empty, undefined is returned.
If the buffer is not empty, the index of the last element in DATA is
cbuf[DATA][(cbuf[BEGIN] + cbuf[POP] - 1) % cbuf[DATA].length]);

cbuf_shift(cbuf) → {Any}

Retrieve and remove the element at the front of the circular buffer (the oldest one in * FIFO) terms). If the operation fails, and this can only happen if the buffer is empty, undefined is returned.
If the operation succeeds and returns a defined object x from index cbuf[DATA][BEGIN], the contents of x's index will be set to undefined. That way, the circular buffer will not be in the way of a possible garbage collection of x
Parameters:
Name Type Description
cbuf CircularBuffer a valid circular buffer

Type

 let z;

 assert(CBUF.last(cbuf) === z);
